2025학년도 제34회 컴퓨터공학과(스마트IT학과) A팀 졸업작품전

    • 작 품 명나만의 가챠샵을 찾는 여정 '가챠로버(Gachalover)'
    • 팀    원김연우, 박채연, 이정은, 이연진
    • 제품기술Android studio, Figma, Firebase
    • 간력한설명내주변의 가챠샵의 위치 공유, 커뮤니티, 가챠 교환까지 모두 통합하여 가챠 생활의 모든 것을 하나로 즐길 수 있는 서비스
    • 작 품 명대학 일정 자동 분류 앱나만의 캠퍼스 비서 '캠비'
    • 팀    원박주희, 이영
    • 제품기술Firebase, Android Studio, Visual Studio Code, Figma
    • 간력한설명자연어로 입력을 통해 대학생의 과목과 학사 일정 유형을 자동으로 분류해주는 스마트 일정 관리 앱
    • 작 품 명AI와 함께 나를 돌아보는 일기 앱 '하루미'
    • 팀    원김은혜, 김희경, 이수경
    • 제품기술Android Studio, Figma, Firebase, Gemini API
    • 간력한설명사용자가 기록한 일기를 AI가 분석해 그래프와 피드백을 제공하고 꾸준히 기록할 수 있도록 식물 성장과 미션이란 게임 요소도 제공하는 앱
    • 작 품 명AI가 함께하는 실전 면접 연습 앱 'JobPilot'
    • 팀    원주서현, 김동미, 권서연, 김주연
    • 제품기술Figma, Android Studio, FireBase, Python
    • 간력한설명취준생을 위한 AI 면접 준비 코칭 앱으로 언제 어디서나 실전같은 연습과 구체적인 피드백으로 면접 실력향상 및 표현과 논리성 개선에 기여
    • 작 품 명지역정보를 한눈에 볼 수 있는 도시정보제공 사이트 '뚜벅뚜벅'
    • 팀    원김서연, 김예지, 윤지민
    • 제품기술php, MSSQL, Visual Studio Code, Figma
    • 간력한설명국내 여행 증가 추세를 기반으로 여러 지역의 축제, 맛집 등 흩어진 여행정보를 한눈에 볼 수 있고 사용자가 편리한 여행일정을 구성할 수 있는 사이트
    • 작 품 명지친 몸을 위해 쉽게 배우는 마사지 앱 ‘리브’
    • 팀    원오현진, 김하늘, 심예은, 윤민
    • 제품기술Android Studio, Figma, Firebase
    • 간력한설명얼굴,어깨 등 몸의 부위별로 다양한 마사지,스트레칭 영상을 제공하여 집에서 쉽고 간편하게 피곤한 몸을 회복할 수 있고 챗봇을 사용하여 1:1 상담을 할 수 있는 마사지 앱
    • 작 품 명IoT를 활용한 모바일 기반 어플 ‘화재 알리미’
    • 팀    원박은진, 박세진, 진유정, 서효희
    • 제품기술IoT,Firebase
    • 간력한설명일산화탄소, 가연성 가스, 온도를 감지해 화재 위험을 조기에 인식하고 앱 푸시 알림으로 알려주는 가정용 화재 예방 알리미
    • 작 품 명아트감상, 거래, 대여 웹사이트 'Artra'
    • 팀    원안지선
    • 제품기술React, Vite, Node.js, MongoDB, RESTful API
    • 간력한설명아트를 감상하거나 홍보하고 사고 팔고 대여할 수 있는 웹사이트
